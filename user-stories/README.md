# User Stories - Airbnb Clone Backend

## Overview

This directory contains user stories for the Airbnb Clone backend project. User stories are derived from the use case diagram and describe the features from the end-user's perspective.

## Contents

- **user-stories.md** - Complete list of user stories with acceptance criteria

## User Story Format

Each user story follows the standard format:

- **As a** [type of user]
- **I want to** [perform some action]
- **So that** [I can achieve some goal/benefit]

## Categories Covered

1. User Authentication
2. Profile Management
3. Property Listings
4. Search and Filtering
5. Booking System
6. Payment Processing
7. Reviews and Ratings
8. Messaging and Notifications

## How to Use

- Review user stories before implementing features
- Use acceptance criteria as a checklist for feature completion
- Update stories as requirements evolve
